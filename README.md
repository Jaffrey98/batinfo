## Installation
```
npm install -g batinfo
```

## Run (after completing above step)
```bash
H:\__tryouts\batinfo>batinfo

🔋  Charge Remaining: 88%

🔌  Charging: YES ⚡

⏲️  Time remaining: Not available

```

## Run (from source)
```
git clone https://www.github.com/jaffrey98/batinfo
```
```
cd batinfo
```
```
node index.js
```
